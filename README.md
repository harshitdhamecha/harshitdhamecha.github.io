# Harshit Dhamecha — Portfolio

**Live → [harshitdhamecha.github.io](https://harshitdhamecha.github.io)**

Personal site of a product-minded AI engineer. I take ambitious AI ideas from a fuzzy whiteboard moment to shipped systems — currently doing exactly that on the Backend AI team at BrowserStack.

## What's inside

- The story: research with a runway, backend with backbone, product partnership that turns ambiguity into roadmap
- A compact timeline of the work, with the receipts (certificates open right on the page)
- My resume, one click away
- A fully playable **vinyl turntable** at the end. Drag the record. You'll hear it.

There are a few more secrets than that. The console knows one of them.

## Built with

Hand-written HTML, CSS and vanilla JavaScript — no framework, no build step. GSAP powers the scroll choreography, the Web Audio API synthesizes the scratch sounds, and the whole thing is served straight from this repo by GitHub Pages.

Designed neo-brutalist on purpose: parchment, ink, acid lime, coral, hard shadows, and type that takes up room.
